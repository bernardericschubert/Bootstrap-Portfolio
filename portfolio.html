JavaScript Psychic Game

This is a game built in JavaScript where a user tests their psychic abilities by attempting to correctly guess a randomly generated letter.

jQuery Emoji Game

This is a game similar to the psychic game, but built in jQuery. Users are given a total value they must meet by choosing 4 emoji images that carry a random point value.

jQuery Trivia Game

This is a trivia game built with jQuery using timers. Users are given 10 seconds to answer a multiple choice question. The app evaluates correct/incorrect answers and a final score.

Giphy API application

This is a jQuery and Ajax application that allows users to query the Giphy API and return images

Train Schedule Admin

This is an admin portal for managing train schedules. The purpose of this application was to work with persistent data storage - in this case Google Firebase. Time manipulation done via Moment.js

Group Project 1

Alpha Cycling is a mobile responsive application built for our first group project. The goal of the application was to provide curated destinations for tourists (or adventurous locals) to visit via bicycle. We provide users the ability to select starting and ending locations, provide information on the locations via multiple APIs, and finally provide directions via the Google Maps API.

Liri Bot

LIRI is a command line interface Node.js app that takes in parameters and gives you back data after querying 3 different APIs. Users are able to query Bands In Town for upcoming concerts, Spotify for track information, and OMDB for movie information.

Bamazon

This is a command line Node.js application built on top of MySql. It allows users to query a database of products, choose one for purchase, and the application decrements the amount in the DB.
